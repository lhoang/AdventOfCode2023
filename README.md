# Advent of Code 2023

https://adventofcode.com/

## Stack

- Node 21
- pnpm (npm compatible)
- Typescript
- Vitest

## Install

```shell
pnpm i

# Install zx for the input fetching script
pnpm i -g zx
chmod +x createDay.mjs
```

## Fetch input and create new file for a daily challenge

```shell
./createDay.mjs
```
